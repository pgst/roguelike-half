```bash
# 全テスト
npm run test:e2e
```
### 1. 特定のテストファイルを指定して実行する
ファイルパスを指定して実行します。
```bash
# npm経由で引数を渡す場合 (  --  を使用):
npm run test:e2e -- tests/skeleton_exchange.spec.ts
# npxで直接実行する場合:
npx playwright test tests/skeleton_exchange.spec.ts
```
### 2. テストのタイトル名（パターン）を指定して実行する
-g （または  --grep）オプションを使用して、特定のテスト名にマッチするものだけを実行します。
```bash
# 例: 「砂掃きの骸骨」が含まれるテストのみを実行する場合
npx playwright test -g "砂掃きの骸骨"
# または、npm経由の場合:
npm run test:e2e -- -g "砂掃きの骸骨"
```
